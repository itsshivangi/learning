# learning
Project Overview

In this project, I began by learning the basics of working with CSV files, followed by building foundational knowledge in Python and the Pandas library. After understanding the core concepts, I gained hands-on experience with commonly used Pandas methods to better analyze and manipulate data.

Currently, I am working with a dataset from the 2018 Central Park squirrel census. I extracted this dataset from an online source and am using it to practice real-world data processing techniques.

## What I’m Doing

•⁠  ⁠Loading and exploring the dataset using Pandas
•⁠  ⁠Cleaning and organizing the data
•⁠  ⁠Analyzing features such as primary fur color and other related attributes
•⁠  ⁠Creating a new CSV file that summarizes key insights from the dataset

## Goal

The final goal of this project is to generate a refined CSV file using Pandas that highlights specific characteristics (such as primary fur color distribution) in a clear and structured format.

This project helps reinforce my understanding of data analysis workflows using Python and Pandas.
